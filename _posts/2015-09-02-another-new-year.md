Today is my birthday and a good day to reflect on how far I've come. My birthday coincides with the start of the academic year, which reinforces the feeling of the start of a new annual cycle. A little over a year ago I entered the Code Oregon program and began an all-consuming quest to change my career. I thought it might be helpful to others embarking on a similar journey to see a timeline of events that brought me to where I am now.

*June 2014 - March 2015:* I joined Code Oregon in June 2014 and bought a MacBook Pro. I studied as much as possible in my free time using Treehouse and various tutorials on the interwebs, but it was hard to make much progress due to the fact that between my full time job and a bus commute, most of my waking hours were spent doing something unrelated to my aspirations. Still, I got a good lay of the web dev land, started attending tech meetups, and became increasingly certain that a career as a developer was the perfect fit for my creative, curious, analytical temperament. I took a summer community college course on Software Design and found a wealth of free educational resources online. I became proficient with the command line and the fundamentals of JavaScript and Python and even participated in a Treehouse pilot program that allowed me to work on a collaborative project with another student using Slack and GitHub.

*April 2015 - May 2015:* I quit my day job in late March and began my full time relentless pursuit to level up my skills. I spent most of April and May studying JavaScript at Code Fellows. I was in heaven.

*June 2015 - August 2015:* I spent all summer studying Python and Django at PDX Code Guild because I wanted more exposure to Python and full stack development. I learned to use Ajax in a single page app, coded a lot in vanilla JavaScript so I could better understand what is happening under the hood of the common JavaScript libraries, made some good friends and contacts, and learned in the process that I like front-end programming a little better than back-end development.

*What's next?* Good question. I'm still sorting out my plan of action. I have a job coach through Code Oregon who has helped me with my resume and LinkedIn profile and has begun to send me job leads. I am continuing to level up my JavaScript skills. Most of the jobs I want require some experience in Angular or React, so I need to add those feathers to my cap in the coming weeks. It wouldn't hurt me to make another Django project so I don't forget all I learned. I still love Python and want to spend more time with it. I also need to obtain a more sophisticated understanding of algorithms and data structures. 

Most importantly, I need to keep building.

In addition to all my web development endeavors, I need to spend more time with my guitars, piano, mandolin, yoga mat, and Italian/Turkish/Icelandic/French studies. These activities are part of a balanced mental diet.  One cannot live on code and caffeine alone. At least not all the time.
