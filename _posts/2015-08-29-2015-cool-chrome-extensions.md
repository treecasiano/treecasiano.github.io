I love finding tools with a low learning curve that make my life as a developer just a tiny bit easier. Here are two of the newest additions to my toolbox.


### Postman - REST Client

[Postman](http://www.getpostman.com/) is a free Chrome extension that allows for convenient testing of your public  API. You can send any HTTP request quickly with their request builder GUI, group your requests with folders, save requests to a history and replay them later, switch contexts with ease, and execute custom scripts. You can write your own tests and even automate them with Postman's testing framework.

I heard about this great extension for the first time today while learning to create my first proper MEAN Stack application with the fantastic and aptly named tutorial ["How to build a M.E.A.N. web application"](https://youtu.be/OhPFgqHz68o) by Travis Tidwell. I highly recommend both the tutorial and the Postman extension.


### JSONView

Speaking of useful Chrome extensions, another new favorite is [JSONView](https://chrome.google.com/webstore/detail/jsonview/chklaanhfefbnpoihckbnefhakgolnmc). JSONView allows you to view JSON files with indentation and syntax highlighting for your viewing pleasure. If you have a Treehouse profile, for example, you can tack .json onto the end of your profile URL and see your account's JSON data. It normally looks like an ugly blob of text, but with the JSONView Chrome extension, it is automatically reformatted. No need to cut and paste it into [JSON Pretty Print](http://jsonprettyprint.com/).
