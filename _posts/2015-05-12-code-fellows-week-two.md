We're already halfway through the third week of my Code Fellows JavaScript course. The material has kept me so busy that I barely have time to write. That's not a complaint, though. I'm having a great time.

The work continues to be challenging. There are times during the class where it feels like if I take my eyes off the screen or allow myself to get distracted for even a moment, I can get a little lost. The pace is rapidfire, though I wouldn't have it any other way. I'm soaking it up as fast as it's coming at me, though perhaps just barely. This pleases me. There's much to gain from learning just at the edge of one's abilities.

The time I invested in learning keyboard shortcuts in Sublime Text and in using the command line has served me well. Even seemingly little things like being able to create files and directories and open them in a code editor from the command line saves valuable time during class, time during which I otherwise might fall behind for a moment and miss a critical instruction. Using multiple cursors and other slick Sublime Text tricks also saves me time and allows me to focus on the instructor's directions and the explanations. Knowing one's tools well is the mark of a professional, and every day I inch a little closer to feeling less like a novice. There still feels like a million miles between me and being a job-ready developer, but I'm enjoying the trip and moving forward every day.

Last week we covered array methods, callback functions, variable scope and hoisting, constructor functions, Object Oriented Programming, and prototypes. We also got more practice writing tests. We continued to use Grunt and Node and became a little more familiar with the Chai assertion library. It's all great fun, and it's keeping me hungry for more.
